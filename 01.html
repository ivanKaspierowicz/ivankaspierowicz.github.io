Diseño de control room

1Q - 2021

Resumen: El presente documento describe el diseño de un pequeño control room para un estudio de grabación. Se realiza bajo el criterio ambechoic, con una capacidad máxima de 8 personas, que se establece en base a las regulaciones del DISPO en CABA, Argentina. Se proponen cinco sistemas de monitoreo distintos: dos para producciones estéreo, uno para producciones binaurales y uno para producciones en audio tridimensional. También son realizados los cálculos para el tratamiento acústico y el aislamiento sonoro. El tiempo de reverberación obtenido para las distintas bandas de frecuencia se encuentra dentro del límite establecido por la AES.



El concepto Ambechoic hace referencia al tipo de ambiente dentro de las habitaciones. Conceptualmente, estos ambientes proveen un entorno acústico uniforme, ideal para sistemas surround. Este concepto fue desarrollado por Petrović y Dadović para habitaciones con un volumen menor a 80 m3. Las condiciones de diseño iniciales son: la difusión homogénea por encima de frecuencias de 1 kHz en todas las superficies (exceptuando el suelo) y la maxima absorción posible por debajo de 250 Hz en todas las superficies excepto el suelo.


Algunas de las herramientas utilizadas para llevar a cabo el diseño de un control room son las simulaciones de BEM (Método de elementos finitos) y de absorción-difusión. Esto con el objetivo de lograr un tiempo de reverberación uniforme para todas las frecuencias. También se emplea el cálculo de las pérdidas por transmisión, a fin de cumplir con los requerimientos de la ley 1540. Además, se realizan análisis de los distintos sistemas de monitoreo disponibles en el mercado y los cálculos de ventilación necesarios para garantizar un bajo nivel de C02 dentro del control room.
